Program leírása: Egy költség követő program, ahol meg lehet adni a dátumot, a költség nevét és az összegét. Ezt utána el lehet menteni és utána bármikor visszatölteni. A program ezt egy diagramban is kijelzi.

Felhasznált modulok:

tkinter: A grafikus felhasználói felület (ablakok, gombok, beviteli mezők) megvalósításához
csv: Az adatok strukturált mentéséhez és beolvasásához
os: Fájlrendszer műveletekhez
datetime: A mai dátum automatikus lekérdezéséhez
matplotlib: A statisztikai kördiagram kirajzolásához.
(Ez külső modil, ha nem működik elöbb telepíteni kell Terminálban a "pip install matplotlib" parancsal)
